# Expense Tracker

A simple and modern **Expense Tracker web app** built using **HTML, CSS, and JavaScript**.  
It helps users track income and expenses month-wise, set spending limits, and visualize expenses using a pie chart.

---

##  Features

-  Add **Income & Expense** transactions
-  **Month-wise tracking** using a month picker
-  **Category-wise pie chart** visualization
-  Data stored in **LocalStorage**
-  Monthly **expense limit warning**
-  **Dark / Light mode** toggle
- **Undo delete** option
-  Keyboard shortcuts support

---

## Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **Canvas API** (for pie chart)
- **LocalStorage**
